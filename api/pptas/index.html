MilMove PPTAS API (0.0.1)

Download OpenAPI specification:Download

E-mail: milmove-developers@caci.com License: MIT

The PPTAS API is a RESTful API that enables the Navy's PPTAS system to request information relating to current moves in progress.

All endpoints are located at /pptas/v1.

moves

PPTASReports

Gets all reports that have been approved. Based on payment requests, includes data from Move, Shipments, Orders, and Transportation Accounting Codes and Lines of Accounting.

query Parameters
since
string <date-time>

Only return moves updated since this time. Formatted like "2021-07-23T18:30:47.116Z"

Responses

http://primelocal/pptas/v1/moves
